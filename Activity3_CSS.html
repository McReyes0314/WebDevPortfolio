Process

In this activity, we are tasked with creating a personal portfolio using Bootstrap components. The portfolio should include the following element: Carousel, Grid Table, and Card.

  • Objective: To Understand how to integrate different Bootstrap elements to build a cohesive and visually appealing web page
  • Instructions:
    1. Create a Navigation Bar
    2. Add a Carousel
    3. Use Grid Layout for skills
    4. Create a Table for Education and Work Experience
    5. Add Cards for Portfolio Projects
    6. Finalize and Style
    7. Test and Refine
    8. Save and Submit
  • Tips
    1. Refer to the Bootstrap documentation for additional guidance and examples.
    2. Use high-quality images for the Carousel and project Cards to enhance the visual appeal.
    3. Keep the design clean and professional, ensuring that the content is easy to read and navigate.

Learnings

Throughout the activity, I learned the following aspects:

  • Created web pages using Bootstrap for default styling and structure.
  • Incorporated various elements, including a carousel, grid table, and card.
  • Experimented with Bootstrap's structure to customize and modify element styling.
