Process

In this activity, we are tasked with creating two simple JavaScript programs: one named menuSelector and the other libraryMenu. These two programs will need to incorporate arrays to store the necessary data and implement a do-while loop to allow users to make choices within the program.

  • Instruction:
    1. Create an Array.
    2. Display Menu.
    3. Use Switch Statement.
    4. Error Handling.
    5. Continue or Exit.

Learnings

Throughout the activity, I learned the following aspects:

  • I learned to problem-solve and analyze how data flows.
  • Apply JavaScript fundamentals for such the use of array for data container.
  • Use a do-while loop to enable user interaction within the console program.
