Process

In this activity, we are tasked to create simple javascript program for ten questions.

  • Instruction: Show your code and output for these following questions:
    1. Calculating Total Cost.
    2. Score Adjustment.
    3. Temperature Conversion.
    4. Inventory Management.
    5. Add Cards for Portfolio Projects.
    6. Age Comparison.
    7. Investment Growth with Monthly Contributions.
    8. Distance Traveled with Varying Speeds.
    9. Enhance Game Scoring System.
    10. Comparative Age Analysis.
    11. Dynamic Countdown Timer with Complex Conditions.

Learnings

Throughout the activity, I learned the following aspects:

  • I learned to problem-solve and analyze how data flows.
  • Apply JavaScript fundamentals to create solutions for each of the following questions.
  • Manipulate data to achieve the desired results for each question.
