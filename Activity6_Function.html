Process

In this activity, we are tasked with creating two JavaScript functions: one named checkAge and the other checkVotingEligibility.

  • Instructions: checkAge Function
    1. If the age is less than 13, return "You are a child."
    2. If the age is between 13 and 19 (inclusive), return "You are a teenager."
    3. If the age is between 20 and 64 (inclusive), return "You are an adult."
    4. If the age is 65 or older, return "You are a senior."
    5. If the input is not a valid number, return "Please enter a valid age."
  • Instructions: checkAge Function
    1. If the age is not a valid number (e.g., negative number or non-numeric), return "Please enter a valid age."
    2. If isCitizen is not a boolean, return "Please specify your citizenship status."
    3. If the age is less than 18, return "You are not eligible to vote."
    4. If the age is 18 or older and isCitizen is true, return "You are eligible to vote."
    5. If the age is 18 or older and isCitizen is false, return "You must be a citizen to vote."

Learnings

Throughout the activity, I learned the following aspects:

  • I learned to problem-solve and analyze how data flows.
  • Apply JavaScript fundamentals for such the use of functions to do a specific task.
  • To return certain result based on the given condition.
