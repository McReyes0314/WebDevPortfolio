Process

Throughout this activity, we are assigned to create web pages for three Philippine holidays, incorporating text and image tags and attributes. Here are the details for our individual activity:

  • Outcome: The output should consist of a basic HTML document that uses text and image tags and attributes effectively, and structures a web page with multiple elements.
  • Instructions:
    1. Create an HTML file for each holiday (e.g., new_year.html, independence_day.html, christmas.html).
    2. Use proper HTML structure.
    3. Insert relevant information for each holiday, including a suitable image in PNG format.
    4. Upload the files.
    5. Submit the link.
  • Content Guidelines:
    1. Include the name of the holiday, the date it is celebrated, and a brief description.
    2. Use high-quality images related to the holiday.
    3. Provide meaningful alt text for each image

Learnings

Throughout the activity, I learned the following aspects:

  • Creating multiple web pages that incorporate the HTML structure.
  • Using text and image elements effectively, including attributes such as alt for images.
  • Adding other elements, such as the anchor () tag, to connect web pages for easy navigation.
  • Utilizing external CSS to design all the web pages by linking them through the element.
